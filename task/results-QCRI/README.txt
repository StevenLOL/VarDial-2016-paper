This package contains your results for the DSL 2016 shared task.

The resources include:

>> results.csv - Results for all of your submissions.

>> results-breakdown.txt - Detailed results for each of your submissions, including precision, recall and per-class performance details.

>> paper-instructions.txt - Useful information for writing your system paper.

>> /paper/ - A directory with a Latex paper template that includes your results. This can be very helpful in getting you started with writing your system description paper.

>> /confusion-matrix/ - A directory with a confusion matrix for each submission, in CSV, TXT, Latex and PDF formats.

>> README.txt - this file
